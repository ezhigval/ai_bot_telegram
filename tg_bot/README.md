## tg_bot — Telegram-бот на Python

Этот подпроект будет содержать Telegram-бота, написанного на Python (рекомендуется `aiogram`), который:

- принимает все типы сообщений от пользователя (текст, голос, фото, видео, документы и т.д.);
- нормализует их в единый формат запроса к ядру (`core/`);
- отправляет запросы в HTTP/WebSocket-API ядра и возвращает ответы пользователю в Telegram.

Бот не реализует ИИ-логику, он только:

- отвечает за устойчивость, логирование, обработку ошибок Telegram API;
- выполняет команды (`/start`, `/help`, `/profile` и т.д.);
- корректно маршрутизирует сообщения в ядро.

Базовый URL ядра (например, `CORE_BASE_URL`) всегда читается из конфигурации/переменных окружения, чтобы:

- при локальном запуске бот стучался на `http://localhost:8000`;
- при выносе ядра на внешний сервер достаточно было сменить конфиг, не меняя код.
