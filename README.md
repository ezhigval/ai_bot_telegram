## AI Telegram Bot (Go)

Локальный Telegram-бот на Go с ИИ-агентом.

### Запуск локально

1. Установи Go (1.21+).
2. Скопируй `.env.example` в `.env` и пропиши переменную:

   - `TELEGRAM_BOT_TOKEN` — токен бота из BotFather.

3. Установи зависимости и запусти:

```bash
cd tg_ai_bot
cp .env.example .env
# пропиши реальный токен в .env (НЕ коммить его в git)
go run ./cmd/bot
```

Бот будет работать по лонг-поллингу и отвечать в том же чате.

Или короче через `Makefile`:

```bash
make run
```

### Hot reload (dev-режим)

Для удобной разработки с авто-перезапуском при изменении кода:

1. Установи `air` глобально (один раз):

```bash
go install github.com/cosmtrek/air@latest
```

Убедись, что `$GOPATH/bin` (обычно `~/go/bin`) добавлен в `PATH`.

2. В корне проекта запусти:

```bash
make dev
```

`air` будет следить за `.go`-файлами и автоматически перезапускать бота при каждом сохранении.

### Структура проекта

- `cmd/bot` — точка входа, запуск Telegram-бота.
- `internal/telegram` — работа с Telegram API.
- `docs` — документация (требования, архитектура, roadmap и т.п.).

### Правила

- Секреты (токены, ssh-ключи) храним только в `.env`, в git не коммитим.
- Вся документация, планы и требования — внутри репозитория (Markdown в `docs`).
