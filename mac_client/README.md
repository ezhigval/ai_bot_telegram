## mac_client — нативный macOS-клиент (Swift/SwiftUI)

Этот подпроект будет содержать:

- Xcode-проект нативного клиента под macOS;
- реализацию экранов и UX (чат, история, работа с медиа/файлами);
- сетевой слой для общения с ядром ИИ (`core/`) по HTTP/WebSocket API.

Клиент:

- не реализует ИИ-логику, только UI/UX и сетевое взаимодействие;
- идентифицирует пользователя по `user_name` (и, при необходимости, доп. ключам);
- использует базовый URL ядра из настроек (никогда не хардкодит адрес сервера).

Перед началом реализации Agent MacClient должен описать структуру экранов и данные,
с которыми они работают, в отдельном документе `docs/mac_client_design.md`.
